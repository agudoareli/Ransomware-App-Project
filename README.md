To execute the program, follow these instructions:
  Go to https://github.com/agudoareli/Ransomware-App-Project 
  Click the green Code button
  Click Download ZIP
  Unzip the folder to a location on your computer 
  Open a terminal 
  In the temnial, navigate to the folder where you unziped the code
    cd Downloads\your-project-folder (windows)
    cd ~/Downloads/your-project-folder (mac)
  Create a virtual environment
    python -m venv gr (windows and mac)
  Activate the virtual environment
    gr\Scripts\activate (windows)
    source gr/bin/activate (mac) 
  Install required packages
    pip install gradio pytrends matplotlib pandas (windows and mac) 
  Run the app
    python main.py (windows and mac)
