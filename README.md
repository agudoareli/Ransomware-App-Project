To execute the program, follow these instructions:
  1) Go to https://github.com/agudoareli/Ransomware-App-Project 
  2) Click the green Code button
  3) Click Download ZIP
  4) Unzip the folder to a location on your computer 
  5) Open a terminal 
  6) In the temnial, navigate to the folder where you unziped the code
    cd Downloads\your-project-folder (windows)
    cd ~/Downloads/your-project-folder (mac)
  7) Create a virtual environment
    python -m venv gr (windows and mac)
  8) Activate the virtual environment
    gr\Scripts\activate (windows)
    source gr/bin/activate (mac) 
  9) Install required packages
    pip install gradio pytrends matplotlib pandas (windows and mac) 
  10) Run the app
    python main.py (windows and mac)
